<color=#800020>Strings</color>	

Strings are sequences of characters that can represent text or data. They are similar to lists in that they are ordered collections, but instead of holding various data types like lists, strings primarily consist of characters. You can perform various operations on strings to manipulate and work with text. Here's how you can work with strings:

CODE
===========================
# Creating a string
my_string = "Hello, World!"

# Accessing characters
first_char = my_string[0]  # Access the first character (index 0)
last_char = my_string[-1]  # Access the last character (index -1)
substring = my_string[7:12]  # Get a substring (index 7 to 11)
print(first_char, last_char, substring)  # H ! World

# String length
length = len(my_string)  # Get the length of the string
print(length)  # 13

# Concatenating strings
string1 = "Hello"
string2 = "World"
concatenated_string = string1 + ", " + string2  # Concatenate two strings
print(concatenated_string)  # Hello, World

# Checking for substrings
contains_hello = "Hello" in my_string  # Check if "Hello" is in the string
print(contains_hello)  # True

# Finding substrings
index_of_comma = my_string.index(",")  # Get the index of the comma
substring_after_comma = my_string[index_of_comma + 2:]  # Get substring after the comma
print(index_of_comma, substring_after_comma)  # 5 World!

# Counting occurrences
count_of_l = my_string.count("l")  # Count the number of times "l" appears
print(count_of_l)  # 3

# Converting to lowercase and uppercase
lowercase_string = my_string.lower()  # Convert to lowercase
uppercase_string = my_string.upper()  # Convert to uppercase
print(lowercase_string, uppercase_string)  # hello, world! HELLO, WORLD!

# Stripping whitespace
whitespace_string = "  This is a string with spaces   "
stripped_string = whitespace_string.strip()  # Remove leading and trailing spaces
print(stripped_string)  # "This is a string with spaces"

# Splitting into a list
word_list = my_string.split(",")  # Split the string at commas into a list
print(word_list)  # ['Hello', ' World!']

# Joining a list of strings
joined_string = ", ".join(word_list)  # Join the list with commas
print(joined_string)  # Hello, World!

# Replacing substrings
new_string = my_string.replace("World", "Python")  # Replace "World" with "Python"
print(new_string)  # Hello, Python!

# Checking for character types
is_alpha = my_string.isalpha()  # Check if all characters are alphabetic
is_digit = my_string.isdigit()  # Check if all characters are digits

# Formatting strings
formatted_string = "Value: {:.2f}".format(3.14159)  # Format a floating-point number
print(formatted_string)  # Value: 3.14
